🌍 ClearEvo.com GIS Converter
→
GIS vector format converter - GDAL ogr2ogr compiled to WebAssembly, runs 100% in your browser (your data is never uploaded). Input: the common vector formats ogr2ogr reads - Shapefile (select .shp + .dbf + .shx + .prj together), GeoJSON, GPX, KML/KMZ (KMZ is unzipped right in the page), GeoPackage, DXF, CSV, MapInfo, FlatGeobuf, GML... Output: GeoJSON, GPX, KML, GeoPackage, CSV, SQLite, DXF, FlatGeobuf. Optional reprojection to any EPSG code. Choose input file(s) to begin. First use downloads the converter engine (~21 MB, cached after).

powered by gdal3.js (GDAL/ogr2ogr in WebAssembly) - 100% in-browser, files never uploaded (Google Analytics: visit stats only - verify) - how to use - more tools - clearevo - © 2026 Kasidit Yusuf, GPL v2
