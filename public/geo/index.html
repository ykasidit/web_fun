GIS vector format converter - GDAL ogr2ogr compiled to WebAssembly, runs 100% in your browser (your data is never uploaded). Input: anything ogr2ogr reads - Shapefile (select .shp + .dbf + .shx + .prj together), GeoJSON, GPX, KML/KMZ, GeoPackage, DXF, CSV, MapInfo, FlatGeobuf, GML... Output: GeoJSON, GPX, KML, GeoPackage, CSV, SQLite, DXF, FlatGeobuf. Optional reprojection to any EPSG code. Choose input file(s) to begin. First use downloads the converter engine (~21 MB, cached after).
